## Overview
The AI Writing Assistant streamlines interactions with generative AI, enhancing task
completion efficiency and making the experience intuitive and valuable for users.

## AI writing assistant for Input fields

It appears within input field component to assist users in creating, iterating,
and improving text entries through Quick Prompts. These instructions for generative
AI are crafted by experts known as prompt engineers, who focus on prompt quality.
The AI Writing Assistant Menu provides writing-specific prompts such as “Change Tone,”
Adjust Length,” Translate,” and “Analyze Text,” which users can apply to the entire text
or a selected portion.
